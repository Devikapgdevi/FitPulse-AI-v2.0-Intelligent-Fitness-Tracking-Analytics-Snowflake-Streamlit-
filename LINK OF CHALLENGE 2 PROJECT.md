## 🚀 About the Project

FitPulse AI is an AI-powered fitness tracking and analytics platform built using **Snowflake + Streamlit**.

The application enables users to:

* 📊 Track and analyze fitness performance in real-time
* 💪 Log workouts with intelligent calorie and XP calculations
* 🤖 Generate personalized workout plans using AI logic
* 🎯 Set and monitor fitness goals with progress insights
* 🧠 Receive adaptive coaching based on user behavior
* 🏆 Engage through gamification (streaks, XP, badges, leaderboard)

## 🔐 Access Instructions

This application is hosted within Snowflake’s Streamlit environment and requires authorized access.

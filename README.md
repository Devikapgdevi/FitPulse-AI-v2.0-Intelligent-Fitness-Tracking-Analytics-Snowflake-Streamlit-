# ⚡ FitPulse AI v2.0

🚀 Intelligent Fitness Tracking & Analytics Platform powered by Snowflake + Streamlit

FitPulse AI is a modern, AI-driven fitness tracking application that helps users monitor workouts, analyze performance, set goals, and receive personalized coaching — all in one powerful dashboard.

---

## 🌟 Features

### 🏠 Dashboard
- Real-time fitness overview
- XP system & streak tracking
- Performance insights & trends
- Motivational quotes

### 💪 Workout Logging
- Log workouts with:
  - Type (Strength, Cardio, HIIT, etc.)
  - Duration & intensity
  - Calories burned (auto-calculated)
  - Mood tracking (before & after)
- Earn XP for every workout

### 🤖 AI Workout Generator
- Personalized workout plans
- Based on:
  - Goal (Fat loss, muscle gain, etc.)
  - Time availability
  - Difficulty level
  - Equipment

### 📊 Analytics
- Workout trends & visualizations
- Calories & duration tracking
- Workout type distribution
- Mood vs performance analysis

### 🎯 Goals & Progress
- Set custom fitness goals
- Track progress with smart indicators
- Health status insights (On Track, At Risk, etc.)

### 🧬 Body Metrics
- Daily check-ins:
  - Weight
  - Sleep
  - Water intake
  - Energy levels
- Trend visualization

### 🧠 Adaptive AI Coach
- Personalized recommendations
- Behavior-based insights
- Smart suggestions based on performance

### 🏆 Gamification
- XP system & levels
- Streak tracking
- Badges & achievements
- Leaderboard ranking

### 📜 History & Export
- Filter workout history
- Export data as CSV

---

## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **Backend**: Snowflake (Snowpark)
- **Language**: Python
- **Database**: Snowflake Data Warehouse
- **Styling**: Custom CSS (Modern UI/UX)
